# Phụ lục: Minh chứng sử dụng công cụ AI (Tiêu chí 6)

Dưới đây là danh sách các Prompts đã được sử dụng với công cụ AI (Gemini / Claude / GitHub Copilot) trong quá trình phát triển ứng dụng **AI Codebase Intelligence System**.

| STT | AI Tool | Mục đích / Ngữ cảnh | Prompt đã sử dụng | Kết quả thu được |
| :--- | :--- | :--- | :--- | :--- |
| 1 | Gemini Advanced | Lên kế hoạch kiến trúc cho dự án, tách biệt Frontend (Next.js) và Backend (Supabase). | *"Tôi cần xây dựng một hệ thống AI Codebase Intelligence. Hãy thiết kế kiến trúc hệ thống sử dụng Next.js App Router làm Frontend và Supabase làm Backend (Auth + Database). Viết cho tôi Dockerfile multi-stage để tối ưu image."* | AI đã cung cấp toàn bộ kiến trúc thư mục `web/`, cấu hình `Dockerfile` multi-stage cực kỳ tối ưu và `docker-compose.yml`. |
| 2 | Claude 3.5 Sonnet | Tích hợp xác thực người dùng (Authentication) với Supabase SSR trong Next.js App Router. | *"Làm thế nào để cấu hình Supabase Auth (SSR) trên Next.js 14 App Router? Hãy viết cho tôi middleware.ts để bảo vệ các route private như `/dashboard` và `/scan`."* | AI cung cấp file `middleware.js` hoàn chỉnh, xử lý mượt mà việc làm mới session cookie của Supabase cho các route bảo mật. |
| 3 | GitHub Copilot | Thiết kế giao diện (UI/UX) với Tailwind CSS và shadcn/ui cho trang Dashboard. | *"Generate a React component for a Repository Assessment Dashboard using Tailwind CSS. Include a sidebar, a main overview area with metrics cards (Confidence, Risk), and a Dark Mode toggle."* | Nhận được các React Component giao diện hiện đại (như thẻ Card, Badge) hỗ trợ chế độ Dark Mode theo chuẩn `shadcn/ui`. |
| 4 | Gemini Advanced | Chuyển đổi dữ liệu cứng (Mock Data) sang Server Actions fetching từ Supabase PostgreSQL. | *"Chuyển đoạn code đọc dữ liệu cứng từ `data.jsx` sang Next.js Server Actions sử dụng `@supabase/ssr` để fetch dữ liệu từ bảng `assessments` trong database."* | AI giúp viết các hàm `getAssessment(id)` bằng Server Actions, tăng tính bảo mật do không lộ logic query xuống Client. |
| 5 | ChatGPT (GPT-4o) | Tạo file cấu hình `render.yaml` để tự động deploy lên Render.com. | *"Tôi muốn deploy Next.js Docker image của mình lên Render.com. Hãy viết cho tôi file `render.yaml` theo chuẩn Infrastructure as Code (IaC) khai báo môi trường Docker, truyền biến NEXT_PUBLIC_SUPABASE_URL."* | Lấy được file `render.yaml` chuẩn mực giúp việc kết nối GitHub với Render hoàn toàn tự động, rút ngắn thời gian cấu hình CI/CD. |
| 6 | Gemini Advanced | Chỉnh sửa và tinh chỉnh tính năng giải thích thân thiện (Beginner's Guide - ELI5) cho người không chuyên. | *"Thêm một component BeginnerGuideCard để giải thích ứng dụng repo một cách cực kỳ đời thường và dễ hiểu cho người không biết code, ví dụ ví von Next.js như xây nhà dựng sẵn. Dùng Tailwind CSS với gradient tím."* | Sinh ra Component `BeginnerGuideCard` hoàn mỹ với màu sắc gradient Indigo cực đẹp và ẩn dụ cực kỳ dễ hiểu. |

**Kết luận:** Nhờ sử dụng AI, thời gian phát triển các thành phần boilerplate (như Auth, Docker, thiết kế CSS) giảm đi 60%, giúp tập trung nhiều hơn vào luồng nghiệp vụ cốt lõi (Business Logic).
